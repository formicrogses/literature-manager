Method 2: Manual API Cleanup (Advanced Users)

If Method 1 is not available, you can use the following code:

Run the following code in your browser console:

// Replace with your GitHub Token
const token = "YOUR_GITHUB_TOKEN_HERE";

// Load cleanup script
const script = document.createElement('script');
script.src = './cleanup-github-repo.js';
document.head.appendChild(script);

// Wait for script to load then run
script.onload = () => {
  const cleaner = new GitHubCleaner(token, "wazgtam1", "literature-data");
  cleaner.cleanupRepository();
};
How to get GitHub Token:
1. Visit GitHub Settings > Tokens
2. Click "Generate new token"
3. Select "repo" permissions
4. Copy the generated token
